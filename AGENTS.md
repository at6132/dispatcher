# Design profile

Dispatcher UI is **liquid glass, low-profile, minimal, easy**.

- Tokens + `GlassSurface`: `src/theme`
- Agent rule (always on): `.cursor/rules/design-profile.mdc`
- No logo chrome. No AI-aesthetic (purple gradients, glow, pill clusters, cosmic dark).
- Cool mist canvas, quiet steel accent, soft blur panels, hairline borders, low texture.
- Prefer clarity and spacing over decoration.
